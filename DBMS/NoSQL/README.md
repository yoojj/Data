# NoSQL
Not Only SQL, NoREL     
: RDB의 기능을 일부 버려(데이터들의 연관성) 처리 속도와 성능을 높임      
: 대규모 데이터(페타바이트 수준)를 유연하게 처리         
: 고속 처리가 필요한 서비스에 사용    
: 인덱스와 데이터가 분리되어 별도로 운영
