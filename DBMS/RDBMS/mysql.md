# MySQL

- [MySQL Command](./mysql-command.md)


```bash
# 설치
$ sudo yum install mysql-server
$ sudo apt install mysql-server

# 설치 확인
$ mysql --version


# 실행
$ sudo systemctl start mysql

# 종료
$ sudo systemctl stop mysql


# 루트 접속
# mysql -h아이피 -u아이디 -p비번
$ mysql -u root -p
$ mysql --user=root -p
```
