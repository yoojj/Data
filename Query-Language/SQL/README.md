**SQL Extension**   
: 벤더사 별 고유 기능이 추가된 SQL           
: RDBMS 제품이 SQL 표준보다 먼저 나와 제품에 따라 차이가 존재      

- PL/SQL (Oracle, MariaDB)
- SQL/PSM (MySQL, MariaDB)
- PL/pgSQL (PostgreSQL)
- T-SQL, Transact-SQL (Microsoft SQL Server, Sybase)
- Watcom-SQL (Sybase)
